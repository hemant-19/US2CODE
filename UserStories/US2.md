# User Story: Feature to calculate loan EMI
**ID:** S001  
**Feature:** F001 - Loan EMI 
**Epic:** E001 - Loan Application  

### User Story
As a **developer**, I want to **create application to calculate loan EMI** so that customer can **calculate his monthly EMI given principal, interest rate, and tenure** .

### Acceptance Criteria
- **AC1**: Application can handle 100 requests per minute
- **AC2**: Application should delete the customer data after customer close the application
